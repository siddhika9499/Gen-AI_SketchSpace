Technical architecture

How it works

01

Canny edge detection

Your sketch is converted to a binary edge map using OpenCV's Canny algorithm, isolating room structure and furniture outlines.

02

ControlNet conditioning

The edge map conditions the diffusion process via lllyasviel/sd-controlnet-canny, ensuring generated images preserve spatial layout.

03

Stable Diffusion v1.5

Style prompts guide denoising over 20–50 steps with UniPC scheduler, producing photorealistic interiors guided by the ControlNet signal.

04

CLIP + Fidelity evaluation

Each output is scored for semantic alignment with the style prompt (CLIP score) and structural preservation relative to the original sketch.
